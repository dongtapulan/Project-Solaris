Project Space.env, formerly known as Project Solaris. 
An immersive, web-based 3D Solar System Explorer built with Three.js and Tailwind CSS. 

This project integrates real-time astronomical data and interactive orbital mechanics to provide a cinematic window into our local group.
 Technical Stack

Engine: Three.js (WebGL)
Styling: Tailwind CSS
Logic: ES6+ JavaScript (Modules)
Data Sources: NASA Horizons & NASA Open APIs


Getting Started
1.  Clone the repository.
2.  Run a local server (e.g., Live Server in VS Code).
3.  Open index.html in a modern web browser.
